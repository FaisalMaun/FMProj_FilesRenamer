📁 File Processor

Smart Rename & Markdown Converter

Welcome! This tool helps you organize your files with intelligent naming and converts their content to structured Markdown format.

  • Upload up to 10 images (JPG, PNG, GIF, WebP) — max 3MB each
  • Upload up to 2 PDF documents — max 3MB each
  • AI-powered description generation for smart file naming
  • Automatic Markdown conversion preserving structure and formatting
  • All files packaged in a single ZIP with new names
  • No files stored — everything processed in memory
